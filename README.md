<h2>🔁 Loops</h2>

<p>
<b>When to use?</b><br>
Use loops when you need to repeat the same task multiple times.
</p>

<pre>
Problem
   │
   ▼
Need repetition?
   │
 ┌─Yes─┐
 ▼     ▼
for   while
</pre>
